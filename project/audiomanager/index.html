Html5 AudioManager

The AudioManager is available in the global scope as a constructor.

This means to access the methods you have to instantiate to a variable using the new keyword eg

{% highlight ruby %} var audiomanager = new AudioManager; {% endhighlight %}

Available Methods

  • play()
  • pause()
  • resume()
  • playSound()
  • createSound()
  • isPlaying()
  • togglePlay()
  • byteFrequencyData()
  • setSCKey()
  • setVolume()
New methods will be added here as I expand the framework.

Basic Usage

{% highlight ruby %} var audiomanager = new AudioManager; var blue = audiomanager.createSound('/mp3/Dead_Combo_-_01_-_Povo_Que_Cas_Descalo.mp3','Dead Combo - Povo Que Cas Descalo'') audiomanager.play(); {% endhighlight %}
